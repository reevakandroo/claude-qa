# PC001: App Loaded at Login URL

| Field | Detail |
|-------|--------|
| **Precondition ID** | PC001 |
| **Title** | App Loaded at Login URL |
| **Summary** | App is open at the login page; no user is logged in |

## Description

The CardioCheck UAT application is open in the browser at the login page. No user is currently logged in.

## Setup Steps

1. Open a web browser (Chrome recommended for Flutter CanvasKit).
2. Navigate to: **https://cardiocheck-uat.up.railway.app**
3. Confirm the login page is fully loaded — **Email** and **Password** fields are visible.
4. Confirm no user is currently logged in (ECG dashboard is NOT shown).
