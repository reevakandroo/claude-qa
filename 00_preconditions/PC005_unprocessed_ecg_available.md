# PC005: Unprocessed (New) ECG Available

| Field | Detail |
|-------|--------|
| **Precondition ID** | PC005 |
| **Title** | Unprocessed (New) ECG Available |
| **Summary** | At least one New ECG entry exists in the list |

## Description

The user is logged in and at least one ECG with "New ECG" status is present in the list.

## Setup Steps

1. Complete [PC003](PC003_user_logged_in_dashboard.md): User is Logged In.
2. Scroll through the ECG list.
3. Confirm at least one **"New ECG"** status entry is present.
4. If none exist, request UAT data be replenished before running these tests.
