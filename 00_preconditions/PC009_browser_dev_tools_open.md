# PC009: Browser Developer Tools Accessible

| Field | Detail |
|-------|--------|
| **Precondition ID** | PC009 |
| **Title** | Browser Developer Tools Accessible |
| **Summary** | Browser DevTools are open |

## Description

Browser developer tools (DevTools) are open for network/console/storage inspection.

## Setup Steps

1. Press **F12** or right-click → **Inspect** to open DevTools.
2. Navigate to the relevant panel as required:
3.   - **Console** — JS errors / logs
4.   - **Network** — HTTP request/response monitoring
5.   - **Application** — cookies, localStorage, sessionStorage
6.   - **Elements** — DOM structure / attribute inspection
