# PC004: ECG List Visible with Entries

| Field | Detail |
|-------|--------|
| **Precondition ID** | PC004 |
| **Title** | ECG List Visible with Entries |
| **Summary** | ECG list is loaded with at least one entry visible |

## Description

The user is logged in and the ECG list has fully loaded with at least one ECG entry.

## Setup Steps

1. Complete [PC003](PC003_user_logged_in_dashboard.md): User is Logged In.
2. Wait for the ECG list to fully load (up to 15 seconds).
3. Confirm at least one ECG entry is visible.
