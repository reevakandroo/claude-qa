# TC119: Validation error messages are human-readable

| Field | Detail |
|-------|--------|
| **Test ID** | TC119 |
| **Module** | User Friendliness |
| **Type** | ✅ Positive |
| **Status** | ⬜ Pending |

## Preconditions

- User submits form with invalid data.

## Execution Steps

| Step | Action |
|------|--------|
| 1 | Enter an age of 0 |
| 2 | Try to submit |
| 3 | Read any validation error message shown |

## Expected Results

Error message is written in plain language such as "Age must be between 18 and 150". No technical codes or jargon shown.
